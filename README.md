# AI-Based Solar Panel Inspection System

This project focuses on detecting visible defects on solar panels using classical computer vision techniques.

The system analyzes images, highlights damaged regions, and estimates defect severity using OpenCV and NumPy.

Main features:
- defect detection
- crack analysis
- severity estimation
- automatic result generation

Technologies:
- Python
- OpenCV
- NumPy

Current limitations:
- small cracks can still be difficult to detect
- reflections may create false positives
- detection quality depends on image conditions

Future improvements may include YOLO-based deep learning detection.
