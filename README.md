# AI-Based Solar Panel Inspection System

This project is an AI-powered solar panel inspection system developed using Computer Vision and YOLOv8.

The system is designed to detect visible defects on solar panels such as:
- cracks
- damaged cells
- surface anomalies
- defective panel regions

The project combines classical computer vision techniques with modern deep learning object detection.

---

## Technologies Used

- Python
- OpenCV
- NumPy
- Ultralytics YOLOv8
- Computer Vision
- Roboflow

---

## Features

- Solar panel defect detection
- YOLOv8 object detection training
- Defect localization with bounding boxes
- Prediction on custom images
- Automatic result generation
- Dataset annotation using Roboflow

---

## Dataset

The complete dataset is not fully uploaded to GitHub due to repository size limitations.

The dataset was created and annotated using Roboflow and contains:

- defective solar panel images
- normal solar panel images
- YOLO format annotations
